# IP Locator Service

REST API сервис на базе FastAPI и PostgreSQL. Приложение определяет город по переданному IP-адресу, сохраняет историю запросов в базу данных и возвращает статистику популярности найденного города.

## Стек технологий

* Python 3.9
* FastAPI
* PostgreSQL 15
* Docker & Docker Compose
* SQLAlchemy

## Требования

* Установленный и запущенный Docker Desktop (или Docker Engine).

## Запуск проекта

1.  Откройте терминал в папке проекта.
2.  Выполните команду для сборки и запуска контейнеров:
    ```bash
    docker-compose up --build
    ```
3.  Сервис будет доступен по адресу `http://127.0.0.1:8000`.

## Использование

### Получение информации о городе
**Метод:** `GET`
**Эндпоинт:** `/city`
**Параметр:** `ip`

**Пример запроса:**
`http://127.0.0.1:8000/city?ip=8.8.8.8`

**Пример ответа:**
```json
{
  "ip": "8.8.8.8",
  "city": "Ashburn",
  "frequency": "Очень много",
  "count_debug": 12,
  "saved_id": 45
}
